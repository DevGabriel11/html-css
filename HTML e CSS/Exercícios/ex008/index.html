Reflexione

Então, ao invés de usar B para "Negrito", é melhor usar STRONG para definir qual parte do texto será mais "Forte".
"Negrito" não tem significado (semântica), apenas refere-se a "Forma", enquanto "STRONG" (Forte), tem significado/semântica;
